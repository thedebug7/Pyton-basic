# Conditionals (if/else)

```
score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
else:
    grade = "C"
print(f"Your grade: {grade}")
```

# Logical Operators (and, or, not)

```
and → Both conditions must be True.
or → At least one condition must be True.
not → Reverses the condition.
```
